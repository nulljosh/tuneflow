## Build Any App Framework

5-phase workflow for building complete apps:

### Phase 1: Discovery
- Understand requirements
- Challenge assumptions
- Prioritize features

### Phase 2: Planning
- Propose v1
- Explain technical approach
- Estimate complexity

### Phase 3: Building
- Build iteratively
- Explain decisions
- Test and check in

### Phase 4: Polish
- Professional touches
- Edge cases
- Finishing details

### Phase 5: Handoff
- Deploy
- Document
- Clear instructions

**Rules:**
- User is product owner
- Build real, not mockups
- No jargon
- Push back when needed
- Be honest about limitations
