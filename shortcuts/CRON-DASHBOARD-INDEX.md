##  Tools

### CLI Tool
**File:** `cron-dashboard` (executable)

Main command-line interface for monitoring cron jobs.

**Usage:**
```bash
cron-dashboard                    # List all jobs
cron-dashboard stats              # Statistics
cron-dashboard watch              # Real-time monitor
cron-dashboard detail "job name"  # View one job
cron-dashboard csv                # Export to CSV
cron-dashboard html ~/report.html # Generate HTML
```

### Web Server
**File:** `cron-web-server` (executable)

Starts an auto-refreshing web dashboard.

**Usage:**
```bash
cron-web-server                   # Start on port 8765
CRON_PORT=9000 cron-web-server   # Custom port
```

**Access:** http://localhost:8765

### Wrapper Script
**File:** `cron` (executable)

Convenience alias for both tools.

**Usage:**
```bash
cron stats                    # = cron-dashboard stats
cron watch                    # = cron-dashboard watch
cron web                      # = cron-web-server
cron detail "job-name"        # = cron-dashboard detail "job-name"
```

## ⚙️ Setup

### Basic (No Setup Needed)
```bash
~/.openclaw/workspace/shortcuts/cron-dashboard list
```

### Add to PATH (Optional)
```bash
export PATH="$PATH:$HOME/.openclaw/workspace/shortcuts"

# Now you can use:
cron-dashboard
cron-web-server
cron
```

### Add Aliases (Optional)
```bash
# Add to ~/.zshrc or ~/.bashrc
alias cron='cron-dashboard'
alias cron-watch='cron-dashboard watch'
alias cron-web='cron-web-server'

# Reload shell
source ~/.zshrc   # or ~/.bashrc
```

##  Pro Tips

**Daily routine:**
```bash
cron-dashboard stats    # 10 seconds, see what's running
```

**Monitor a job:**
```bash
cron-dashboard watch 3  # Updates every 3 seconds
```

**Share a report:**
```bash
cron-dashboard html ~/report.html
# Email the HTML file
```

**Backup jobs:**
```bash
cron-dashboard json > ~/cron-backup-$(date +%Y%m%d).json
```

**Quick filter:**
```bash
cron-dashboard json | jq '.[] | select(.enabled == true) | .name'
```

##  Quick Help

**Command not found?**
```bash
# Use full path:
~/.openclaw/workspace/shortcuts/cron-dashboard

# Or add to PATH:
export PATH="$PATH:$HOME/.openclaw/workspace/shortcuts"
```

**Web server won't start?**
```bash
# Check if port is in use:
lsof -i :8765

# Try different port:
CRON_PORT=9000 cron-web-server
```

**Gateway not running?**
```bash
# Check status:
openclaw gateway status

# Should show: Runtime: running
```

**Can't find a job?**
```bash
# Use partial name (case-insensitive):
cron-dashboard detail "morning"

# Or full job ID:
cron-dashboard detail "12ca412a-1252-46bb-819d-eac84bc46412"
```
