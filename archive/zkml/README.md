# ZKML Tools

Zero-Knowledge Machine Learning analysis tools.

## Files

- `zkml_tool.py` - Main ZKML tool
- `zkml_analyzer.py` - Analysis script
- `zkml_deep_analyze.py` - Deep analysis variant
- `zkml_compare.py` - Comparison tool v1
- `zkml_compare2.py` - Comparison tool v2
- `zkml_compare3.py` - Comparison tool v3
- `ZKML_FORMAT.md` - Format specification

## Usage

```bash
python3 zkml_tool.py
python3 zkml_analyzer.py
python3 zkml_compare.py
```

See `ZKML_FORMAT.md` for format specifications.
