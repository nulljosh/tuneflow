# Graphics Generators

80s-style graphics generators using HTML Canvas and Node.js.

## Files

**HTML Generators:**
- `80s-flying-name.html` - Flying text animation
- `80s-name-image-generator.html` - Name to 80s graphic

**JavaScript Generators:**
- `generate_80s_birthday.js` - Birthday card generator
- `generate_80s_birthday_clean.js` - Clean birthday card variant
- `generate_80s_gameover.js` - Game over screen
- `generate_80s_image.js` - Generic 80s image
- `generate_80s_radical.js` - Radical style
- `generate_80s_retrowave.js` - Retrowave aesthetic
- `generate_birthday_cake.js` - Birthday cake graphic
- `generate_grandma_*.js` - Various grandma-themed graphics (arcade, beach, cyberpunk, graffiti, perfection, rockstar)

**Python Generators:**
- `generate_80s_name.py` - 80s name graphic via Python

**Generated Images:**
- All PNG files with 80s and grandma themes

## Usage

Run Node.js generators:
```bash
node generate_80s_birthday.js
```

Run Python generator:
```bash
python3 generate_80s_name.py
```

Open HTML generators directly in browser.
